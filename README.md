# Lemon case study

**Case Study**

**First Stage**  
Implement a catapult entirely in C++ (no Blueprints allowed).  
Pressing the **FIRE** button in the UI should spawn a physics-enabled projectile.  
This projectile must detach from the catapult and follow a uniform linear motion, reaching at least **Y height** and **X distance**, both of which must be configurable from the code.

Showcase your best skills in this exercise. This includes unit testing, YAGNI, KISS, dependency injection, git strategy, MVC, MVVM, MVP, Decision Tree, FSM, anything you are proud of. The goal is not to finish the case study, is to showcase your skills.
